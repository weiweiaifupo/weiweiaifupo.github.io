# vue 组件开发

# 按钮大小

选用 small 大小如：

<el-button size="small">小型按钮</el-button>

# 表单选项

选用 small 大小,添加自适应布局容器，如：

<el-form
  ref="form"
  label-position="left"
  :model="formData"
  label-width="80px"
  size="small"
  class="form-modification"
>
  <el-row :gutter="0">
    <el-col :span="7">
      <el-input></el-input>
    </el-col>
  </el-row>
</el-form>

# 标题字体大小

有全局样式，设置 class 名即可，如：

<div class="body-title">版本管理</div>

或单独设置

font-size: 16px;
font-family: PingFangSC-Medium, PingFang SC;
font-weight: 600;
color: #182a4e;

# 表格大小

# 表格分页

统一为每页 10、30、50、100，默认 30

# 表格单元格高度

表头与表格主体单元格都是 40px----(全局样式，不需要单独设置，在开发过程中留意一下即可，看样式是否生效；有操作列的表格，文字按钮会把表格撑开,需要修改文字按钮样式)

# 开发要求

1、单个组件总代码尽量不要超过 300 行

2、每天开发前先拉取最新代码

3、上传代码前先执行 npm run lint

4、页面文件放在views文件夹下边，公共组件位置放在components文件夹

5、文件名小驼峰命名

component: () => import('@/views/Home/index.vue')

    UiTable,
  },

    return {
      // heightNumber:400,

          label: '标准编号',

        },
        {
          label: '标准名称',
          prop: '',
        },
        {
          label: 'ABC标识',

        },
        {
          label: '标准状态',
          prop: '',
        },
        {
          label: '操作',
          attrs: { width: '100px' },

            const isDelete =
              scope.row.isPublish === 0 ? (
                <el-button type="text" onclick={() => this.deletes(scope.row)}>
                  删除
                </el-button>
              ) : null
            return (
              <div class="" style="display:flex;justify-content:center;">

        },
      ],
    }
  },
}
</script>

# 树形数据下拉框

参数说明类型
value/v-model绑定的label值String/Number
code树形数据的orgCode值String/Number
label树形数据的label值，与v-model相同String/Number
treeId树形数据的id值String/Number
treeAttrs树形组件的属性Object
treeEvents树形组件的事件Object
other用法与element原来的属性一样Any
